## Overview

This repository contains a Power BI report analyzing five years of sales, customer and operational data for Four Clothing .CO (FY 2021–2025). The working report (`Four Clothing.pbix`) and supporting export (`Four Clothing.pdf`) live in this folder.

Key dashboard pages: Executive Summary, Product Performance, Customer Insights, Location Analysis, Growth & Scalability, Strategic Summary.
